# jsQuestPlus

This folder vendors the ES module build from `jsquest-plus@2.1.0`.

- Package: https://www.npmjs.com/package/jsquest-plus
- Repository: https://github.com/kurokida/jsQuestPlus
- License: MIT, copied in `LICENSE`

The static asset is committed so static-hosted experiments can import Quest+
without an npm build step.
